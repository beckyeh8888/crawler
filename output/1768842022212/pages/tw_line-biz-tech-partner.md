## 更多 LINE 功能服務  
助力診所提升官方帳號經營成效

![LINE 微型官網](https://alleypin.com/images/element/e-08.svg)

#### LINE 微型官網

讓 LINE 官方帳號變身成為診所形象官網，透過「圖文選單」讓服務項目一覽無遺、直覺點選，並串接 HIS，實現診所營運的數位化。

![LINE 通知型訊息](https://alleypin.com/images/element/e-09.svg)

#### LINE 通知型訊息

即使不是 LINE 好友也能透過 LINE 通知型訊息 API 發送通知。一旦手機號碼在資料庫比對成功，即使從未加入好友也能收到提醒訊息！

![LINE Bot 客服機器人](https://alleypin.com/images/element/e-10.svg)

#### LINE Bot 客服機器人

AlleyPin 為 LINE 外掛模組服務商，能協助您透過 LINE 聊天機器人來實現任何診所醫病對話場景，讓民眾看診體驗「有感」升級。

![LINE 預約看診服務](https://alleypin.com/images/element/e-11.svg)

#### LINE 預約看診服務

引導顧客加入診所 LINE 好友或完成手機綁定，即可提供 LINE 線上掛號、候號查詢、改期或取消，以及更多便民服務。

![LINE 自動化約診提醒](https://alleypin.com/images/element/e-12.svg)

#### LINE 自動化約診提醒

無論從櫃檯輸入至 HIS 完成約診或自行於 LINE 上掛號，都能自動推送 LINE 約診提醒訊息。有效減輕櫃檯負擔、降低爽約率。

![LINE 官方帳號代營運](https://alleypin.com/images/element/e-13.svg)

#### LINE 官方帳號代營運

從品牌建立策略、醫客溝通定調到實際內容企劃，AlleyPin 專業顧問團隊將協助您一步步打造品牌口碑，創造營運成長的正循環！
